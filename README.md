# twitter-race
Python - twitter hashtag tracker [console based]
Keep track of different hashtags and see who is winning

# Install

`pip -r requirements.txt`

# Usage
`track.py <hashtag#1> <hashtag#2> ... <hashtag#n>`

# Output Example
```
##########################
# python      #   00019   #
# javascript  #   00046   #
# ruby        #   00011   #
##########################
```
